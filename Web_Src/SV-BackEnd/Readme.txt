How to set up with pycharm comunity
Version: Python(3.10)

Steps to set up:
1. open pycharm (open the folder as project then down all python stuffs as recommended from pycharm)
2. open terminal in pycharm
3. cd to SV-BackEnd
4. pip install -r requirements.txt
5. cd to mqtt_sv
6. run server: python manage.py runserver --noreload
